## FizzBuzz

Create a function that takes a number as input.

Print out all numbers between 0 and the given number.

If the number is dividable by 3 print fizz instead.

if the number is dividable by 5 print buzz instead

If the number is dividable by both 3 and 5 print FizzBuzz instead.
